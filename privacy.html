Политика конфиденциальности

Дата последнего обновления: 18 июля 2026 года.

Настоящая политика относится к локальному инструменту se-publisher, который используется владельцем проекта «сё» для подготовки и публикации материалов в принадлежащих владельцу аккаунтах социальных платформ.

se-publisher является локальным инструментом владельца проекта «сё». Он не предоставляет сервис другим пользователям и используется только для публикации материалов в аккаунты, принадлежащие владельцу.

Какие данные используются

Для подключения к Pinterest приложение использует авторизацию OAuth. Доступ к Pinterest предоставляется только после OAuth-разрешения владельца аккаунта. Приложение может получать технические сведения, необходимые для работы с аккаунтом владельца, включая список принадлежащих ему досок и идентификаторы созданных публикаций.

Как используются данные

Данные используются исключительно для чтения принадлежащих владельцу досок, выбора доски, публикации одобренных владельцем Pins и получения результата публикации. Приложение не используется для рекламы, профилирования пользователей, перепродажи данных или публикации от имени третьих лиц.

Хранение данных

Токены доступа и иные учётные данные хранятся только локально на устройстве владельца и не публикуются на сайте. Сайт не содержит API-ключи, токены или секреты. Проект не продаёт и не передаёт персональные данные третьим лицам.

Отзыв доступа

Владелец аккаунта может отозвать доступ приложения через настройки Pinterest. После отзыва доступа приложение больше не сможет обращаться к аккаунту через API.
